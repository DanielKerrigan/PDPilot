
pdp_ranking
=====================================

Version: |release|

A custom Jupyter Widget Library for ranking and visualizing partial dependency plots


Quickstart
----------

To get started with pdp_ranking, install with pip::

    pip install pdp_ranking

or with conda::

    conda install pdp_ranking
